# File Integrity Monitor 🔐

A Python cybersecurity tool that detects unauthorized 
file modifications using SHA-256 hashing.

## 🔍 What It Does
- Calculates SHA-256 hash of any file
- Saves original hash permanently to disk
- Detects even a single character change
- Prints ALERT/SAFE threat report with timestamp
- Monitors multiple files in one session

## 🚀 How To Run
1. Make sure Python is installed
2. Download mainn.py
3. Open terminal and run:
python mainn.py
4. Enter any file path to monitor
